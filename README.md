# 🌍 Carbon Footprint Awareness Platform

An AI-powered sustainability dashboard built using Streamlit.

## 🚀 Features

- 🌱 Carbon Footprint Calculator
- 📈 Trend Analysis with Plotly
- 🤖 AI Recommendations using Gemini
- 📄 PDF Report Generation
- 🎯 CO₂ Reduction Goals
- 🌳 Tree Equivalent Counter
- 🏆 Eco Badge System
- 🌙 Green Dark Theme UI

## 🛠 Technologies Used

- Python
- Streamlit
- Plotly
- Pandas
- Gemini API
- FPDF
- SQLite

## ▶ Run Locally

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
streamlit run app.py
```
